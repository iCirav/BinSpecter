# BinSpecter
![BinSpecter Logo](/assets/images/logo.png)

BinSpecter is an advanced hex editor extension for Visual Studio Code. It allows you to inspect and edit files in hexadecimal format directly within your editor, making it ideal for reverse engineering, binary patching, or analyzing large binary files.

## Features

- View and edit files in hexadecimal format
- Search, replace, and navigate binary data
- Support for large files
- Seamless integration with VS Code
- User-friendly interface for developers and reverse engineers

## Installation

1. Open Visual Studio Code.
2. Go to the Extensions view (`Ctrl+Shift+X` or `Cmd+Shift+X` on macOS).
3. Search for `BinSpecter` and install the extension.
4. Reload VS Code to activate the extension.

## Usage

- Open any file in VS Code.
- Use the command palette (`Ctrl+Shift+P` or `Cmd+Shift+P`) and type `BinSpecter: Open in Hex Editor`.
- Navigate, edit, and save your changes.
