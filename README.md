# BinSpecter

![BinSpecter Logo](/assets/images/logo.png)
BinSpecter is an advanced hex editor extension for Visual Studio Code, designed to let developers inspect and edit binary files directly inside the editor.

## Features

- View and edit files in hexadecimal format
- Search, replace, and navigate binary data
- Supports large files efficiently
- Seamless VS Code integration

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/iCirav/BinSpecter.git
   cd BinSpecter
```
